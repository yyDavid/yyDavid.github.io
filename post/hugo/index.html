Hugo

June 30, 2019

ubuntu18.04 安装hugo

安装hugo

sudo apt install hugo

生成site目录

hugo new site blog

安装主题

git clone https://github.com/budparr/ .git themes/ananke

装饰站点

  • 将主题示例内的static、content文件夹和config.toml文件替换到站点，略作修改即可

局域网内预览博客

hugo server -t ananke --buildDrafts --bind 192.168.2.200 --baseURL http://192.168.2.200 hugo server 命令可以使用 hugo 内置服务器调试预览博客 --theme 选项可以指定主题 --buildDrafts 包括标记为草稿（draft）的内容

新建文章

hugo new hugo.md

部署到 GitHub

  • 在githua创建仓库 yydavid.github.io //yydavid为github昵称小写

hugo --theme=ananke --baseUrl=https://yydavid.github.io --buildDrafts //生成public目录
cd public
git init
git add -A
git commit -m “hugo.md”
git remote add origin https://github.com/yyDavid/yydavid.github.io.git
git push -u origin master
